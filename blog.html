What is event loop? and how does it work?

What is event loop?

In computer programming, an event loop is a programming
construct or design pattern that waits for and
dispatches events or messages in a program

how does event loop work?

The event loop in JavaScript is a fundamental concept
that determines how asynchronous code is executed. Here
is an overview of how it works:

The JavaScript runtime maintains a call stack, which keeps track of function calls in the program.
When a function is called, it is added to the call stack. When the function returns, it is removed from the call stack.
Asynchronous tasks, such as I/O operations or timer events, are executed separately from the call stack in the background.

Cookies

Cookies are small data files that are stored on a computer or mobile device
when you visit a website. These files contain
information such as login credentials, preferences,
and browsing activity. Cookies are created by websites and
sent to a user's web browser, which stores the information for future use.

Difference between map, filter,find and foreach in javascript?

Map in javascript

In JavaScript, map is a higher-order function that creates
a new array containing the results of calling a
provided function on every element in the original array.

Filter in javascript

In JavaScript, the filter() method is used to create a new array
containing all elements of an existing array that pass
a specific test provided by a function

Find in javascript

The find() method in JavaScript is an array method that
searches an array for the first element that satisfies
a given condition and returns that element.r>

Foreach in javascript

forEach is a method in JavaScript that is used to loop
through an array and execute a function on each element of the array.
